Privacy Policy for Content LLM Analyzer

Last updated: February 1, 2026

Overview

Content LLM Analyzer is a Chrome extension that helps you analyze how search engines and AI systems interpret your web content. This privacy policy explains what data we collect, how we use it, and your rights.

Data Collection

The extension collects the following data only when you explicitly click the extension icon:

  • Page title
  • Meta description
  • Page URL
  • Heading structure (H1-H6 tags)
  • Page text content (up to 20,000 characters)
  • Introduction/excerpt (first ~150 words)

The extension does NOT:

  • Track your browsing history
  • Collect data automatically or in the background
  • Access pages unless you activate the extension
  • Collect personal information (name, email, etc.)

How We Use Your Data

Extracted content is sent to our analysis API (content-llm-analyzer.vercel.app) where it is:

  • Processed using Google Cloud Natural Language API for category detection
  • Analyzed using Anthropic Claude API for content clarity recommendations
  • Not stored permanently - data is processed in memory and discarded
  • Not shared with third parties beyond the AI service providers (Google, Anthropic)
  • Not used for training AI models

Third-Party Services

We use the following third-party services to analyze content:

  • Google Cloud Natural Language API - for content categorization (see Google's Privacy Policy)
  • Anthropic Claude API - for content analysis (see Anthropic's Privacy Policy)

Local Storage

The extension uses Chrome's local storage to:

  • Temporarily pass extracted data to the analysis application
  • Store your optional API key preferences (if you choose to provide your own Google/Anthropic keys)

All data stored locally remains on your device and is not transmitted anywhere except when you explicitly run an analysis.

Permissions Explained

  • activeTab - Read page content when you click the extension icon
  • scripting - Extract structured content from pages (including JavaScript-rendered content)
  • storage - Store API key preferences locally on your device

Your Control

You have complete control:

  • The extension only runs when you click the icon
  • You choose which pages to analyze
  • You can uninstall the extension at any time
  • Uninstalling removes all locally stored preferences

Data Retention

We do not retain your content data. Analyzed content is processed in real-time and immediately discarded after analysis results are returned to you.

Children's Privacy

Our extension is not intended for children under 13. We do not knowingly collect data from children.

Changes to This Policy

We may update this privacy policy from time to time. The "Last updated" date at the top will reflect any changes. Continued use of the extension after changes constitutes acceptance of the updated policy.
